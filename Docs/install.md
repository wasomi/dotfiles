# install

1. `sudo pacman -S --needed git`
2. `git clone https://github.com/wasomi/dotfiles ~/.dotfiles --depth 1`
3. `cd ~/.dotfiles`
4. `./install.sh`

# some useful commands
1. `hyprctl monitors` - shows all available monitors and their parameters
2. `hyprctl devices` - shows all available devices (mice, keyboards, etc.) and their parameters
